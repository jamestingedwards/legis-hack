A format to make legislation readable…

…For computers

The AsciiDoc format is plain text with some simple markup. Using plain text makes it easy to track changes, and do other transformations, with tools familiar to most programmers.

…For people

Raw AsciiDoc is still pretty readable for people. That means the automatic comparison of “what changed” is more useful than with raw XML. AsciiDoc also renders to look like a formatted web page, so the text is still approachable for non-technical users.
